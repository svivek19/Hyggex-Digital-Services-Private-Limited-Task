## Run Locally

Clone the project

```bash
  git clone https://github.com/svivek19/Hyggex-Digital-Services-Private-Limited-Task.git
```

Go to the project directory

```bash
  cd Hyggex-Digital-Services-Private-Limited-Task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
